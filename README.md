# Marcelo Augusto — Portfólio Web

Site pessoal de portfólio de **Marcelo Augusto**, desenvolvedor web especializado em criação de sites modernos, rápidos e focados em resultado.

🌐 **Acesse online:** [portifolio-marcelo-augusto.vercel.app](https://portifolio-marcelo-augusto.vercel.app)

---

## Sobre o projeto

Site desenvolvido do zero, sem templates, com foco em performance e conversão. Apresenta os projetos realizados, serviços oferecidos, depoimentos de clientes reais e formulário de contato integrado ao WhatsApp.

## Seções

- **Hero** — apresentação com chamada principal
- **Portfólio** — projetos em vídeo (advocacia, massoterapia, transportadora e portfólio pessoal)
- **Serviços** — abordagem e diferenciais
- **Depoimentos** — clientes reais
- **FAQ** — dúvidas frequentes
- **Contato** — formulário com envio direto via WhatsApp

## Tecnologias

- HTML5, CSS3, JavaScript puro
- Google Fonts (Barlow Condensed + Outfit)
- Vercel (hospedagem)
